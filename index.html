2D Breakout Game

How To Play

  • Press Start Game Button
  • Use Arrow Keys To Move Left & Right
    or
    Use Mouse To Move Left & Right
  • Enjoy The Game!!

    • Rules

  • You Have 3 Lives
  • If The Ball Touches The Bottom You lose 1 Life
  • Destory All Tiles to Win!!
    • About This Project

    I created this project, with the help of a MDN tutorial, learned a lot about building a game from ground up using pure JavaScript and I also created a very basic UI using Bootstrap and basic CSS. This was a fun project, I learned a lot and had a blast at the same time..
